# Сравнение алгоритмов сортировки

Проект демонстрирует реализацию и сравнение трёх классических алгоритмов сортировки: пузырьком, выбором и вставками. Код написан на языке Python.

## Функциональность

Программа выполняет следующие задачи:
* реализует три алгоритма сортировки с возвратом отсортированного списка без изменения исходного;
* тестирует время выполнения каждого алгоритма на случайных данных;
* строит график зависимости времени выполнения от размера входных данных;
* выводит таблицу с результатами замеров времени.

## Алгоритмы сортировки

1. **Сортировка пузырьком** (`bubble_sort`):
   * оптимизирована — останавливается, если на проходе не было обменов;
   * временная сложность: $O(n)$ в лучшем случае, $O(n^{2})$ в среднем и худшем.

2. **Сортировка выбором** (`selection_sort`):
   * на каждом шаге находит минимальный элемент и ставит его на текущую позицию;
   * всегда выполняет $O(n^{2})$ операций независимо от входных данных.

3. **Сортировка вставками** (`insertion_sort`):
   * эффективна для почти отсортированных массивов;
   * временная сложность: $O(n)$ для отсортированных данных, $O(n^{2})$ для случайных и обратных.
